# Тестовое на Python, Farpost 2026
По заданию было не очень понятно как работать с комментариями, поэтому в базу данных Blogs добавил таблицу comment с полями id, author_id(FK на author), post_id(FK на post), text.
Реализован небольшой GUI при переходе на `/`, далее вводится имя пользователя, и можно выполнять различные действия с базой данных
## Датасеты
Датасеты получаем при помощи GET запроса на `/comments?login=Имя пользователя` и `/general?login=Имя пользователя`.
Датасеты возвращаются в JSON формате
В *comments*:
* login - логин пользователя оставившего комментарий
* header - заголовок поста
* post_author - логин автора поста
* comments - число комментариев оставленных данным пользователем к этому посту
В *general*:
* date - дата
* logins - число входов на сайт
* logouts - число выходов с сайта
* actions - число действий внутри блога, в них включенны действия в пространтсве blog и post
